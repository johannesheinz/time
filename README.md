# Time

A simple static webpage that displays the current time.

## Color themes

 - `O` **O**riginal (grey)
 - `B` **B**lack
 - `W` **W**hite
 - `D` Solarized **D**ark
 - `L` Solarized **L**ight

## Goals

 - Keep it simple
 - Use local time from the user's time zone
 - Work in as many browsers as possible, e.g. IE8
